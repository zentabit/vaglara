
Geometrical Optics Tool

Program skapat av Samuel Selleck för Lunds Universitet,
för frågor kontakta sellecksamuel@gmail.com



en fil config.txt behöver finnas i samma directory som programmets jar fil
för att programmet ska starta. Ett exempel på hur config filen kan se ut:

-----------------------config.txt---------------------
Screen properties:
fullscreen=false
width=1200
height=700

LightProperties:
minwavelength=400
maxwavelength=700
defaultwavelength=700

Performance callibration:
opticsobjectresolution=300
maxraybounce=10
colormoderaycount=10


Admin:
admin=false
----------------------/config.txt---------------------
